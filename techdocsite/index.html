HTML Elements

An HTML element is defined by a start tag, some content, and an end tag. Here are some other examples of HTML elements: <h1>This is a heading</h1>, <div>This is a division element</div>, <span>This is a span element</span>, <a href="https://example.com">This is a link</a>. <img src="image.jpg" alt="This is an image">

CSS Syntax

A CSS rule-set consists of a selector and a declaration block. Here are some other examples of CSS rules: h1 { color: blue; }, p { font-size: 16px; }, .container { width: 100%; }, #header { background-color: #333; .btn { padding: 10px 20px; }

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task. Here are some other examples of JS functions: function myFunction() { return "Hello World!"; }, function add(a, b) { return a + b; }, function greet(name) { return "Hello, " + name; }, function multiply(a, b) { return a * b; } function divide(a, b) { return a / b; }

React Components

React components let you split the UI into independent, reusable pieces. Here are some other examples of React components: function Welcome(props) { return <h1>Hello, {props.name}</h1>; }, function App() { return <div><Welcome name="Sara" /><Welcome name="Cahal" /><Welcome name="Edite" /></div>; }, function Clock(props) { return <div><h1>Hello, world!</h1><h2>It is {props.date.toLocaleTimeString()}.</h2></div>; }, function Avatar(props) { return <img className="Avatar" src={props.user.avatarUrl} alt={props.user.name} />; } function Comment(props) { return <div className="Comment"><UserInfo user={props.author} /><div className="Comment-text">{props.text}</div><div className="Comment-date">{formatDate(props.date)}</div></div>; }
